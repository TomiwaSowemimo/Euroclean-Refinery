# Euroclean Refinery & Petrochemical Nigeria Limited

Official website for Euroclean Refinery and Petrochemical Nigeria Limited — a greenfield crude oil refinery development project located in Warri Delta State Nigeria.

## About the Project

This website was built to represent Euroclean Refinery's online presence, showcasing their products, services, and investment opportunity.

## Built With

- HTML5
- CSS3
- JavaScript
- Google Fonts
- Pexels for images

## Features

- Full screen image slider hero section
- Animated scroll reveal effects
- Animated stat counters
- Responsive mobile design
- Interactive FAQ accordion
- Parallax background images
- Card hover effects
